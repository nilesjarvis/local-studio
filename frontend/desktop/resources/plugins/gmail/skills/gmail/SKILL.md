# Gmail

Use `search_threads` to find conversations with Gmail query syntax. Use `get_thread` for a conversation, `get_message` for one exact message, and the list tools only when their inventory is needed.

Keep searches narrow, summarize private content only for the requested task, and never imply that read-only tools sent, deleted, labeled, or modified mail.
